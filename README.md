# 📖 הלימוד היומי - הוראות העלאה

## 🎯 שלב 1: יצירת ריפו חדש ב-GitHub

1. היכנסי ל: https://github.com/new
2. **Repository name:** `limud-yomi`
3. **Public** (חובה כדי לפרסם!)
4. ✅ **Add a README file**
5. לחצי **"Create repository"**

## 🎯 שלב 2: העלאת הקבצים

1. בריפו החדש - לחצי **"Add file"** → **"Upload files"**
2. גררי את הקבצים:
   - `index.html`
   - `service-worker.js`
   - `manifest.json`
3. ב-**"Commit changes"** → **"Commit"**

## 🎯 שלב 3: הפעלת GitHub Pages

1. **Settings** (ריפו הגדרות)
2. בצד שמאל → **Pages**
3. תחת **"Source"**:
   - Branch: **main**
   - Folder: **/ (root)**
4. **Save**
5. חכי 2-3 דקות
